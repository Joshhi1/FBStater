## Documentations

## Introduction:
FBStater gets your appState for logging in at facebook-chat-api!

## How to use:
Add your email / password at config/config.json.

```json
{
  "account": {
    "email": "example@email.com",
    "password": "mycoolpassword"
  }
}
```

After adding your info's, just run it!
```
npm run start
```
